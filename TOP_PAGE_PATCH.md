# トップページへの追加パッチ（手動貼り付け用）

今回、既存サイト `https://furima.ryukyu-tane.com/` のソースファイルやCMS構造には
直接アクセスできなかったため（閲覧のみでファイル構成不明）、安全のため
**既存トップページ本体には一切手を加えていません。**

代わりに、追加したいHTMLの断片と挿入位置をここにまとめます。
サイト管理者ご自身、または実装担当の方が、下記を該当箇所に貼り付けてください。

---

## 追加するリンクカード（共通パーツ）

```html
<a class="ai-editor-card" href="/ai-editor/">
  <strong>AI創作編集室を体験する</strong>
  <p>
    作品が生まれ、出展につながるまでを、動く編集室で体験できます。
    2027年2月頃、田場公民館で次回開催を企画中。
    出展、来場、ボランティアに興味がある方もこちらから。
  </p>
</a>
```

推奨スタイル（既存サイトのCSSファイルに追記、または `<style>` として貼り付け）：

```css
.ai-editor-card {
  display: block;
  max-width: 560px;
  margin: 24px auto;
  padding: 18px 20px;
  border: 2px solid #B4472F;
  border-radius: 14px;
  background: #FFF7E9;
  color: #3A2E22;
  text-decoration: none;
}
.ai-editor-card strong {
  display: block;
  font-size: 1.05rem;
  color: #8F3521;
  margin-bottom: 6px;
}
.ai-editor-card p {
  margin: 0;
  font-size: 0.92rem;
  line-height: 1.7;
}
.ai-editor-card:hover,
.ai-editor-card:focus-visible {
  border-color: #8F3521;
  background: #FFF0D8;
}
```

## 挿入位置1：トップの主要ボタン付近

現在トップページ冒頭には、見出し「沖縄文芸フリマ」の直後に
「当日の様子を見る」リンクと「公式インスタグラム」リンクが並んでいます。

**この2つのリンクのすぐ下**（同じセクション内、次の見出し `## ✨ 第2回開催レポート` の前）に、
上記のリンクカードを1つ挿入してください。

イメージ：

```
[当日の様子を見る] [公式インスタグラム]

<!-- ここに ai-editor-card を挿入 -->

## ✨ 第2回開催レポート (2025.12.13)
```

## 挿入位置2：「沖縄文芸フリマとは？」の直前

見出し `## 🌸 沖縄文芸フリマとは？` の**直前**に、同じリンクカードをもう1つ挿入してください。

イメージ：

```
（第1回開催の様子セクションの終わり）

<!-- ここに ai-editor-card を挿入 -->

## 🌸 沖縄文芸フリマとは？
```

## 正式な申込・お問い合わせURLが決まったら（差し替え箇所）

現時点では、出展・来場・ボランティアの正式な受付フォームURLが確定していないため、
`ai-editor/index.html` 内の3つの入口ボタンと「お問い合わせ」リンクは、
すべてページ内の `#contact`（お問い合わせ方法セクション）へのリンクにしてあります。

正式なフォームやお問い合わせページのURLが決まったら、`ai-editor/index.html` 内の
以下の箇所を実際のURLに差し替えてください（`href="#contact"` となっている4箇所）。

1. カード1「出展に興味がある」ボタン
2. カード2「開催情報を受け取りたい」ボタン
3. カード3「ボランティアに興味がある」ボタン
4. ページ下部CTAの「お問い合わせ」リンク

用途ごとに別々のフォーム（出展用／来場情報登録用／ボランティア用）を用意する場合は、
それぞれのボタンのリンク先を個別のURLに変更してください。
URLのハッシュやクエリでの自動入力は、実際のフォームがその機能に対応している場合のみ使用してください。

## 注意事項

- リンク先は相対パスで `/ai-editor/`（サイトのルート直下に `ai-editor` フォルダを配置する想定）としています。
  もしサイトの公開ディレクトリ構成が異なる場合は、`href` を実際のパスに合わせて調整してください。
- 既存サイトがCMS（WordPress等）やテンプレートエンジンを使っている場合、
  上記のHTML断片は「本文エリア」に埋め込む想定です。テーマファイルを直接編集する場合は、
  必ず編集前にファイルのバックアップを取ってください。
- 貼り付け後は、実機またはブラウザの開発者ツールでスマートフォン幅（375px）表示を確認してください。
